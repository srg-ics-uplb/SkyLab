SkyLab

A Web Application for vCluster

  • Login
  • View on Github

Deploy an MPI Cluster
on demand

This is achieved by utilizing vCluster, a feature of Peak-Two Cloud (P2C). P2C is an Openstack-based private cloud on top of commodity hardware targeted for scientific and high-performance computing.

Manage your own cluster

You can view your cluster status, activate tools or share your cluster to others.

Execute tasks
after a few clicks

No need to open a terminal and do all the dirty work.

Monitor tasks

You can easily keep track of your tasks. You can view your task details and current progress.

Visualize your task output

Compatible output files are rendered with JSmol.
